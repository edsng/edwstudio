# edwstudio.com

Freelance web development portfolio showcasing 7 fully interactive demo websites built with React, TypeScript, and modern web technologies.

## Live Site

[edwstudio.com](https://edwstudio.com)

## Demo Sites

- **FlowSync** — B2B SaaS landing page with interactive dashboard, GSAP scroll animations, parallax effects, and animated counters
- **Diamond Touch Detailing** — Premium auto detailing site with gallery, multi-step quote modal, and calendar booking
- **Marquez Home Group** — Real estate site with searchable listings dashboard, property modals, and neighborhood guides
- **Forge Performance Coaching** — Fitness coaching site with scroll-linked 3D stats panel and multi-step booking
- **Evergreen Outdoor Design** — Landscaping site with GSAP scroll-animated pavers, parallax sections, and project gallery
- **Crown & Fade Barber Lounge** — Barbershop site with cut gallery, barber profiles, and appointment booking
- **Radiant Skin Aesthetics** — Med spa site with treatment cards, provider credentials, and consultation booking

## Stack

- React 19
- TypeScript
- Vite
- GSAP + ScrollTrigger
- React Router
- CSS (no framework)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deploys to GitHub Pages via GitHub Actions on push to `master`.
